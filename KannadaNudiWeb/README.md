# Kannada Nudi Editor (Web)

This is the web version of the Kannada Nudi Editor, built with Blazor WebAssembly and Syncfusion Word Processor.

## Features
- Rich Text Editing
- Kannada Transliteration (Nudi/Baraha style)
- Speech-to-Text (Web Speech API)
- Unicode/ASCII Conversion

## Prerequisites
- .NET 8 SDK

## How to Run
1. Navigate to the `KannadaNudiWeb` directory.
2. Run `dotnet run`.
3. Open the browser at the specified URL (usually http://localhost:5xxx).
