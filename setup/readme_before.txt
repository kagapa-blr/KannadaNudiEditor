BEFORE INSTALLING

Thank you for choosing to install this KAGAPA software.

Please read the following before continuing:

1. Close any other running applications.
2. Make sure you have permission to install software on this computer.
3. If your antivirus shows a warning, allow the installer to run.
4. The installer will copy files to the Program Files folder and
   create shortcuts in the Start Menu.
5. If you already have an older version, it is better to uninstall it first.

Click "Next" to continue the installation.
